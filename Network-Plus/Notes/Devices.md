# Network Devices

## Routers
- **OSI Layer 3 - routes traffic with IP**
- **Inside of switches(L2) - layer switch 3 device**
- **Examples:** LAN, WAN, copper, fiber

## Switches
- **OSI Layer 2(except for layer switch 3 device)**
- **Hardware that bridges devices**
- **PoE(Power over Enthernet):** Enthernet + Power

## Firewalls
- **Filters traffic**
- **Can be either a physical device or built-in a router**
- **Ingress/Egress**

## IDS/IPS (Intrusion Detection/Prevention System)
| Feature | IDS (Detection) | IPS (Prevention) |
| :--- | :--- | :--- |
| **Main Function** | Monitoring network traffic for vulnerabilities | Monitoring network traffic for vulnerabilities |
| **Action** | Detects and **alerts** but does not block. | Detects and **blocks** the attack. |
| **Analogy** | A security camera (records/alarms). | A security guard (stops the intruder). |
